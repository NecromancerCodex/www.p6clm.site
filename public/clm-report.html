Concept

CLM이란? — 건설 AI 운영체제

흩어진 현장 업무(계획·시공관리·문서·감찰)를 하나의 AI 에이전트가 끌고 갑니다. 핵심은 두 가지 — ① 건설 지식을 담은 온톨로지(AI의 기억), ② 그 기억을 검색해 일하는 스타 에이전트 Cbot.

Cbot — 온톨로지를 물고 있는 스타 에이전트

사용자의 요청(글·사진·음성·전화·무전)을 분석해 알맞은 작업에 분배(오케스트레이션)하고, 매 단계 건설 온톨로지를 검색해 근거를 제시합니다.

온톨로지 = AI의 외부 기억 — 표준품셈·시방서(KCS)·법령·과거 공정표를 구조화해 저장. LLM이 외우는 게 아니라 여기서 꺼내 씀 → 환각 억제
에이전틱 서치 — LLM이 단계마다 스스로 이 기억을 2회+ 검색해 근거 축적
멀티모달 — 텍스트·이미지·음성·전화·무전 입력을 모두 이해
휴먼 인 더 루프 — AI가 판단근거를 적고, 사람이 검토·수정 후 확정
Use Cases · 실제 화면

CLM 활용 사례 — 실제 화면으로

제품의 실제 동작 화면입니다.

1

멀티모달 스타 에이전트 — Cbot

무엇이든 묻고, 사진·음성·전화·무전까지 입력. Cbot이 요청을 분석해 맞는 작업으로 보내고 KCS/KDS 근거로 답합니다.

멀티모달오케스트레이션KCS/KDS 근거
▲ AI 대화 화면 — 이미지·음성 입력 가능, ConstructBot이 KCS/KDS 기준으로 답변
2

BIM → 자동 공정표 생성

IFC(BIM)를 공종별로 올리면 공종을 자동 판정하고, 온톨로지 에이전틱 서치(표준품셈·시방서·법령·과거 공정표)로 액티비티를 정의해 WBS·공정표를 만듭니다.

온톨로지 에이전틱 서치 2회공종 자동판정WBS 405 · 구역 26
▲ (좌) 공종별 멀티 임포트 + AI 판정 — 토목·구조·건축·MEP·조경·가설 / (우) WBS 워크패키지 405개·구역 26개 자동 도출
3

AI 판단근거 작성 + 휴먼 인 더 루프

AI가 "왜 이렇게 계획했는가"를 적습니다. 사람이 활동·기간·선후행을 직접 수정·삭제한 뒤 컨펌하면 공정표 베이스라인이 됩니다.

AI 판단근거사람이 검토·수정액티비티 2124개
▲ 액티비티 정의 — "AI 판단 근거: 왜 이렇게 계획했는가" + 활동별 직접 수정/삭제 (휴먼 인 더 루프)
4

스케줄링 리스크 분석 + 베이스라인 근거

CPM + 자원 평준화로 날짜를 계산하고, 목표공기 초과·우기 토공 등 리스크를 자동 진단합니다. 베이스라인이 왜 이렇게 나왔는지 근거를 제시하고 AI가 브리핑합니다.

리스크 자동 진단AI 브리핑베이스라인 근거
▲ 리스크 분석(목표공기 초과·우기 토공) + AI 브리핑 + 베이스라인 근거 + 간트 차트
5

프리마베라(P6) 호환 + 내역서로 정밀화

생성된 공정표는 P6 XML로 내보내 Primavera에서 그대로 임포트·수정할 수 있습니다. 기존 도구를 버리지 않아도 됩니다.

P6 XML 호환임포트·수정 가능

💡 내역서(BOQ)를 넣으면 얼마나 정밀해지나?

구분현재 (BIM만)내역서 추가 시
기간 산정부재 수·연면적 기반 추정실제 물량(콘크리트량·거푸집·철근) 기반 정밀 산정
원가미산정물량 × 표준품셈 단가 → 공정별 원가
활동 상세공종·층 단위부재별 자재·수량까지 (콘크리트 종류, 거푸집, 철근)

→ BIM = "무엇이 어디에", 내역서 = "얼마나 · 얼마짜리". 둘을 합치면 기간·원가가 동시에 정밀해집니다. (연동 훅 준비됨, 샘플 입수 시 적용)

6

4D 시뮬레이션 — 오늘 날짜 진행을 육안으로

공정표와 BIM 부재를 연결해, 슬라이더 날짜에 어디까지 지어졌는지 색으로 표시합니다. 골조가 층별로 올라가고 토공이 깊이별로 진행되는 게 눈에 보입니다.

공정표↔부재 연결날짜별 진행 시각화24,136개 100% 매칭
▲ 4D 대시보드 — 공정 2,124건 · 부재 24,136개 100% 매칭, 부재 클릭 시 공정·기간·상태 표시
7

관리자 워크 + 향후 IoT 결합 — 현장 안 가도 감찰

가상 현장을 1인칭으로 걸어다니며 내부를 점검합니다. 향후 IoT·CCTV·센서와 결합하면 관리자가 직접 가지 않아도 실시간 현장을 감찰할 수 있습니다.

관리자 워크(1인칭)실사 모드향후 IoT 연동
▲ 관리자 워크 ON · 실사 모드 — 가상 현장 내부를 걸으며 골조·부재 점검
8

멀티모달 AI 보고서 작성

Cbot이 요청(사진 + 프롬프트 + 전화·무전 내용)을 분석해 알맞은 보고서를 작성합니다. 온톨로지 에이전틱 서치로 관련 법안·시방서를 근거로 달고, 당연히 수정 가능합니다.

사진+프롬프트 분석법안·시방서 근거전화·무전 → 문서수정 가능
▲ Cbot이 입력(사진·음성·전화·무전)을 분석해 NCR·안전·공사일보 등 문서를 자동 작성 — 온톨로지 근거 포함
9

토공·지반 모델링 — 도면을 데이터로

add 기능으로 도면의 시추공·대지경계선·CIP·Pile을 CSV로 추출·임포트하면, 3D 지층 모델과 층별 토공 물량·단면도가 자동으로 나옵니다.

도면 → CSV 추출3D 지층 모델층별 토공 물량
▲ (좌) 시추 11공 IDW 보간 3D 지층 모델 / (우) 층별 토공 물량 표(토사·풍화·암반) + 지질 단면도
