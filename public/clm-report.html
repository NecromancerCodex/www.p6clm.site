Concept

두 가지가 특별합니다

일반 AI와 다릅니다.
① 멀티모달 — 사진·음성·전화·무전을 다 알아듣고,
② 건설 기준을 갖춤 — 표준품셈·시방서·법령을 찾아보고 정확하게 답합니다.

하나의 에이전트가 현장 전체를 돕습니다

멀티모달 — 글·사진·음성·전화·무전, 무엇으로 시켜도 알아듣습니다.
건설 기준을 갖춤(온톨로지) — 표준품셈·시방서·법령·과거 공정표를 정리해두고, 답할 때마다 찾아봐 근거 있게 답합니다.
한 번에 처리 — 공정표·4D·보고서·현장 감찰까지 한 AI가 처리합니다.
사람이 검토 — AI가 초안을 만들면, 사람이 확인·수정해 확정합니다.
Use Cases · 실제 화면

활용 사례 — 실제 화면으로

제품의 실제 동작 화면입니다.

1

사진·음성·말을 사람처럼 알아듣고 실행

무엇이든 묻고, 사진·음성·전화·무전으로 시키세요. 에이전트가 사람처럼 알아듣고 맞는 작업을 알아서 실행합니다.

멀티모달사진·음성·전화·무전건설 기준 근거
▲ "너는 뭘 할 수 있니?" — AI가 할 수 있는 일을 안내 (공정표·4D·보고서·법령검색·토공). 사진·음성·전화·무전 입력 가능
2

사진 한 장으로 보고서 자동 작성

현장 사진 + 한마디(예: "안전점검보고서 작성해줘")면 보고서가 자동으로 만들어집니다. 관련 법령·시방서를 근거로 쓰고, 당연히 수정도 됩니다.

사진+한마디 → 문서법령·시방서 근거전화·무전 → 문서수정 가능
▲ (좌) 사진 첨부 + "NCR보고서 작성해줘" → 자동 작성 / (우) 완성된 보고서 — 기준·내용·조치를 건설 기준 근거로 작성
3

BIM 파일만 올리면 공정표가 자동

BIM(설계 모델)을 올리면 공종을 자동으로 나누고, 건설 지식(표준품셈·시방서·법령·과거 공정표)을 찾아보며 공정표를 만듭니다.

BIM → 공정표 자동공종 자동 분류건설 기준 근거내역서 연동
▲ (좌) 공종별로 BIM 올리면 AI가 토목·구조·건축·설비 자동 분류 / (우) 공정 범위(작업 405개·구역 26개) 자동 도출

💡 내역서를 함께 올리면 — 기간·원가까지 정밀

구분BIM만내역서까지
기간대략 추정실제 물량(콘크리트·거푸집·철근) 기반 정밀
원가없음물량 × 단가 → 공정별 원가

→ BIM은 "무엇이 어디에", 내역서는 "얼마나·얼마짜리". 둘을 합치면 기간과 원가가 함께 정밀해집니다.

4

왜 이렇게 짰는지 AI가 설명 + 사람이 검토

AI가 "왜 이렇게 계획했는지" 근거를 적어줍니다. 사람이 작업·기간·순서를 직접 수정·삭제한 뒤 확정합니다.

AI가 근거 설명사람이 검토·수정
▲ "AI 판단 근거: 왜 이렇게 계획했는가" + 작업별 직접 수정/삭제 (사람이 최종 확인)
5

공사 위험을 AI가 미리 경고

공기 초과·우기 토공 같은 위험을 AI가 자동으로 찾아 알려주고, 왜 위험한지·어떻게 대응할지 브리핑합니다.

위험 자동 진단AI 브리핑
▲ 위험 분석(공기 초과·우기 토공) + AI 브리핑 + 공정표(간트)
6

프리마베라(P6)에서 그대로 사용

만든 공정표는 P6 파일로 내보내 Primavera에서 바로 열고 수정할 수 있습니다. 쓰던 도구를 안 바꿔도 됩니다.

P6 호환바로 수정
7

오늘 어디까지 지어졌는지 눈으로

공정표와 BIM을 연결해, 날짜를 움직이면 그날까지 지어진 부분이 색으로 보입니다. 골조가 층층이 올라가는 게 한눈에 보입니다.

날짜별 진행 시각화공정표↔BIM 연결
▲ 4D 시뮬레이션 — 부재를 누르면 공정·기간·상태 표시 (전 부재 매칭)
8

현장에 안 가도 가상으로 둘러보기

가상 현장을 1인칭으로 걸어다니며 내부를 점검합니다. 앞으로 IoT·CCTV와 연결하면 실시간 원격 감찰까지 가능합니다.

가상 현장 워크향후 IoT 연동
▲ 가상 현장 내부를 걸으며 골조·부재 점검
9

도면을 올리면 3D 지층·토공 물량

도면의 시추공·경계선 등을 올리면 3D 지층 모델과 토공 물량이 자동으로 나옵니다.

도면 → 3D 지층토공 물량 자동
▲ (좌) 시추 자료로 만든 3D 지층 모델 / (우) 층별 토공 물량 + 지질 단면도
